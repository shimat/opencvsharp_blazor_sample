`haarcascade_frontalface_default.xml` is from [OpenCV](https://github.com/opencv/opencv)'s own `data/haarcascades/` directory (BSD 3-Clause license), used by the Live Face Detection sample.

`face_detection_yunet_2023mar.onnx` is the [YuNet](https://github.com/opencv/opencv_zoo/tree/main/models/face_detection_yunet) face detector from [OpenCV Zoo](https://github.com/opencv/opencv_zoo), used by the DNN Face Detection sample; see [`face_detection_yunet_2023mar.LICENSE`](face_detection_yunet_2023mar.LICENSE) (MIT, Copyright (c) 2020 Shiqi Yu).
